```

        ↓

        ↓
3. Chat GPT 5.5 (мозг операции) — на основе правила передаёт запрос
   в нужный инструмент на обработку
        ↓
4. Выбранный инструмент обрабатывает запрос
        ↓

        ↓

   подготавливает к выдаче
        ↓
7. Агент передаёт готовый результат в чат Telegram в удобном читаемом виде
```

## АГЕНТ РАЗВЕДЧИК — детальное описание

**Тип узла:** Агент 1 — Агент разведчик и сбора информации

**Мозг агента:** Open Router + Chat GPT 5.5

**Функции:**
1. Исходя из запроса — понять какой ресурс и модель лучше подходят для задачи
2. Провести разведку, сбор данных и анализ под задачу
3. На основе анализа — либо предоставить информацию, либо подобрать подходящий контент
4. Выбирать подходящую модель/ресурс для просмотра и анализа информационных ресурсов, сайтов и соц. сетей
5. Выбранная модель должна уметь:
   - Просматривать весь вид и тип контента
   - Распарсить контент и содержимое
   - Просматривать статьи, текст, посты, PDF, картинки
   - Определять нужные ссылки и куда они ведут

## РОЛИ ИНСТРУМЕНТОВ

| Инструмент | Роль |
|---|---|
| **Open Router** | Мозг и управление агентом — банк моделей, выбор ресурса под задачу |
| **Chat GPT 5.5** | Мозг логики — понимает смыслы, обрабатывает результаты, выдаёт итог |
| **Apify** | Парсинг видео и соц. сетей, поиск трендового контента |
| **Whisper** | Транскрибация видео — перевод видео в текст |
| **Firecrawl** | Парсинг информации и контента с сайтов, детальный разбор |
| **Perplexity** | Проверка достоверности, разведка по бизнес-запросам и фактам |
| **LLM Chat GPT 5.5** | Изучение смыслов видео + генерация идей на основе трендов |

## АГЕНТ ОБРАБОТЧИК (Агент транскрибации видео)

**Входящие данные:** видео или список ссылок на видео (после Apify)

**Правила работы:**
1. Транскрибировать видео
2. Смотреть визуал и находить хуки:
   - Цепляющие крючки
   - Офферы
   - Текст, смысл и подача
3. Присылать итоговый отчёт:
   - Что именно выстреливает
   - Что является вирусным и хайпом
   - Наиболее интересное для общей массы и ЦА на данный момент

**Формат выхода:** отчёт в Telegram

## ПРАВИЛА ФАЙЛА "Алгоритм работы.md"

> Правила маршрутизации — какой инструмент вызывать при каком запросе:

| Ситуация | Инструмент |
|---|---|
| Запрос связан с бизнесом пользователя* | **Perplexity** |
| Нужно работать со смыслом и логикой | **Chat GPT 5.5** |
| Нужно проверить контент на достоверность | **Perplexity** |
| Запрос на парсинг информации с сайта | **Firecrawl** |
| Работа с трендами, соц. сетями, поиск видео | **Apify** |
| Выбран вариант "создание контента" | Сначала запускается алгоритм поиска и анализа трендов |

*Бизнес пользователя: заработок на партнёрских программах + инвестиции в криптовалюту на 2 площадках.

> Список правил будет дополняться и расширяться в ходе разработки.

## ПРАВИЛО ОБРАБОТКИ — поиск смысла и хайпа

**Условие:** Поступает запрос на поиск смысла, подбор идеи или нахождение релевантного/выстрелившего контента, набравшего популярность и хайп.

**Цепочка инструментов:**
```
1. Open Router + Chat GPT 5.5 — определяет какая модель лучше подходит
       ↓
2. Apify — парсит видео и соц. сети
       ↓
3. Firecrawl — парсит текстовый контент и сайты
       ↓
4. Chat GPT 5.5 — изучает смыслы транскрибированного видео
       ↓
5. Perplexity — делает разведку и проверяет факты
       ↓
6. Chat GPT 5.5 — присылает итоговый вариант в соответствии с запросом
```
