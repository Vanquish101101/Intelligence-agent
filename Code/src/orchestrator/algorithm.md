# Алгоритм работы — Агент-разведчик

## Правила маршрутизации инструментов

### Когда использовать Perplexity:
- Поиск актуальных фактов, новостей, цифр
- Проверка достоверности информации с источниками
- Быстрый ответ на бизнес-вопрос (крипта, маркетинг, партнёрки)
- Когда нужна актуальность (последние 24–72 часа)

### Когда использовать Firecrawl:
- Парсинг конкретного сайта или статьи по URL
- Получение полного текстового содержимого страницы
- Анализ конкурентного сайта

### Когда использовать Apify:
- Тренды в YouTube (популярные видео, быстрорастущий контент)
- Парсинг соц. сетей (TikTok, VK, Telegram каналы)
- Когда нужны метрики контента: просмотры, лайки, скорость роста

### Когда использовать Whisper:
- Видео без субтитров — нужен полный текст для анализа
- Анализ авторского стиля, структуры подачи по аудио
- Fallback: сначала попробовать YouTube subtitles через Apify

## Стратегия обработки запроса

### Тип "report" (полный отчёт):
1. Perplexity → базовый контекст и актуальные факты
2. Apify → YouTube тренды по теме
3. Firecrawl → поиск по тематическим сайтам
4. Агрегация всех данных
5. LLM → анализ + формирование структурированного отчёта

### Тип "trends" (быстрые тренды):
1. Perplexity → что сейчас в тренде
2. Apify → YouTube топ видео за неделю
3. LLM → краткая сводка

### Тип "search" (поиск по теме):
1. Perplexity → поиск с источниками
2. (опционально) Firecrawl → углублённый парсинг

## Fallback стратегия

| Инструмент | Если недоступен | Действие |
|---|---|---|
| Perplexity | Единственный результат — Apify/Firecrawl | Пометить в отчёте |
| Apify | Нет данных соц. сетей | Пропустить блок, добавить примечание |
| Firecrawl | Нет веб-парсинга | Пропустить, использовать только Perplexity |
| Whisper | Нет транскрибации | Использовать YouTube subtitles через Apify |

## Правила формирования отчёта

1. Отчёт всегда на русском языке
2. Конкретика: числа, названия, ссылки — лучше чем общие слова
3. Структура строго по шаблону из ТЗ
4. Объяснять ПОЧЕМУ залетело — хук, триггер, эмоция, тайминг
5. Идеи для контента — конкретные, применимые сразу
6. Если инструмент не сработал — честно указать в разделе ⚠️
