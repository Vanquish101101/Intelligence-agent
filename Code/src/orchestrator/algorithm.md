# Алгоритм работы — Агент-разведчик

## Глобальное правило: режим deepresearch

**Все запросы на анализ и поиск выполняются в режиме глубокого исследования (deepresearch).**
По умолчанию используется `depth: standard`. При выборе `depth: deep` система переключается в режим максимальной детализации:
- Perplexity: модель `sonar-pro` вместо `sonar`, до 1800 токенов, с system-промптом аналитика
- Apify: до 10 видео вместо 5
- Firecrawl: до 8 результатов вместо 5
- LLM: модель `claude-sonnet-4-6` вместо `claude-haiku-4-5`, до 3000 токенов
- В отчёте: 5+ трендов, детальный разбор по каждой платформе, 5+ идей

## Правила маршрутизации инструментов

### Когда использовать Perplexity:
- Поиск актуальных фактов, новостей, цифр
- Проверка достоверности информации с источниками
- Быстрый ответ на бизнес-вопрос (крипта, маркетинг, партнёрки)
- Когда нужна актуальность (последние 24–72 часа)
- В deep-режиме: используй sonar-pro и расширенный промпт

### Когда использовать Firecrawl:
- Парсинг конкретного сайта или статьи по URL (команда /scrape)
- Получение полного текстового содержимого страницы
- Анализ конкурентного сайта
- Поиск по тематическим статьям (в составе /report)

### Когда использовать Apify:
- Тренды в YouTube (популярные видео, быстрорастущий контент)
- Парсинг соц. сетей (TikTok, VK, Telegram каналы)
- Когда нужны метрики контента: просмотры, лайки, скорость роста

### Когда использовать Whisper:
- Видео без субтитров — нужен полный текст для анализа
- Анализ авторского стиля, структуры подачи по аудио
- Fallback: сначала попробовать YouTube subtitles через Apify

### Когда использовать OSINT-модуль:
- Поиск информации о конкретном человеке, компании, никнейме (/osint)
- Разведка конкурентов по домену или email
- Анализ публичных профилей в социальных сетях
- Категории OSINT: SOCMINT (соцсети), Domain/IP, Email, Username, Company, Phone
- Инструменты: Perplexity sonar-pro (deep web search) + Firecrawl (прямой скрейпинг профилей)

## Правило обработки: запросы на смысл, идею, вирусный контент

*Если поступает запрос связанный с поиском смысла или подбором идеи или нахождением релевантного и выстрелившего контента набравшего популярность и хайп — агент понимает какой инструмент и в какой последовательности задействовать из имеющихся для того чтобы правильно определить смыслы, и подобрать варианты для создания похожего контента.*

Цепочка инструментов для таких запросов:

1. *OpenRouter + ChatGPT 5.5* — определяет какая модель лучше подходит, выстраивает логическую цепочку работы
2. *Apify* — смотрит соц. сети, парсит видео
3. *Firecrawl* — детальный парсинг информации с сайтов
4. *ChatGPT 5.5* — изучает смыслы транскрибированного видео
5. *Perplexity* — делает разведку и проверяет факты
6. *ChatGPT 5.5* — присылает итоговый вариант в соответствии с запросом

## Стратегия обработки запроса

### Тип "report" (полный отчёт):
1. Perplexity → базовый контекст и актуальные факты
2. Apify → YouTube тренды по теме
3. Firecrawl → поиск по тематическим сайтам
4. Агрегация всех данных
5. LLM → анализ + формирование структурированного отчёта

### Тип "report" в deep-режиме:
1. Perplexity sonar-pro → глубокий анализ с расширенным промптом
2. Apify → YouTube топ-10 видео за неделю
3. Firecrawl → 8 тематических статей
4. Агрегация с увеличенным контекстом
5. claude-sonnet-4-6 → детальный отчёт (5+ трендов, 5+ идей)

### Тип "trends" (быстрые тренды):
1. Perplexity → что сейчас в тренде
2. Apify → YouTube топ видео за неделю
3. LLM → краткая сводка (quick-режим)

### Тип "search" (поиск по теме):
1. Perplexity → поиск с источниками
2. (опционально) Firecrawl → углублённый парсинг

### Тип "osint" (разведка по цели):
1. Определить тип цели: username / email / domain / person / company / phone / ip
2. Perplexity sonar-pro → глубокий поиск по открытым источникам (SOCMINT, GEOINT)
3. Для username: Firecrawl → прямая проверка профилей на VK / Telegram / GitHub
4. Для domain: Firecrawl → скрейпинг сайта
5. Форматирование OSINT-отчёта

## Fallback стратегия

| Инструмент | Если недоступен | Действие |
|---|---|---|
| Perplexity | Единственный результат — Apify/Firecrawl | Пометить в отчёте |
| Perplexity sonar-pro | Fallback на sonar standard | Понизить качество, не падать |
| Apify | Нет данных соц. сетей | Пропустить блок, добавить примечание |
| Firecrawl | Нет веб-парсинга | Пропустить, использовать только Perplexity |
| Whisper | Нет транскрибации | Использовать YouTube subtitles через Apify |
| claude-sonnet (deep) | Fallback на claude-haiku | Понизить модель, сохранить функционал |

## Правила формирования отчёта

1. Отчёт всегда на русском языке
2. Конкретика: числа, названия, ссылки — лучше чем общие слова
3. Структура строго по шаблону из ТЗ
4. Объяснять ПОЧЕМУ залетело — хук, триггер, эмоция, тайминг
5. Идеи для контента — конкретные, применимые сразу
6. Если инструмент не сработал — честно указать в разделе ⚠️
7. В deep-режиме: расширять каждый блок, давать дополнительный контекст
8. Стоимость запроса всегда передаётся в meta.cost_usd — отображается пользователю
